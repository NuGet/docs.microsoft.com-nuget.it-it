## <a name="usage"></a><span data-ttu-id="6f93e-112">Utilizzo</span><span class="sxs-lookup"><span data-stu-id="6f93e-112">Usage</span></span>

<span data-ttu-id="6f93e-113">dove `<packageID>` denomina il pacchetto da installare (usando la versione più recente) o `<configFilePath>` identifica il file di `packages.config` che elenca i pacchetti da installare.</span><span class="sxs-lookup"><span data-stu-id="6f93e-113">where `<packageID>` names the package to install (using the latest version), or `<configFilePath>` identifies the `packages.config` file that lists the packages to install.</span></span> <span data-ttu-id="6f93e-114">È possibile indicare una versione specifica con l'opzione `-Version`.</span><span class="sxs-lookup"><span data-stu-id="6f93e-114">You can indicate a specific version with the `-Version` option.</span></span>
