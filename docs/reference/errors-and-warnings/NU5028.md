# <a name="nuget-error-nu5028"></a>Errore NuGet NU5028
<pre>No project was provided to the PackTask.</pre>

### <a name="issue"></a>Problema

Nessun file di progetto specificato per l' `msbuild -t:pack` operazione.


### <a name="solution"></a>Soluzione

Specificare il progetto da inserire nell'operazione di Pack.  Ãˆ possibile specificarlo in Cammand ( `msbuild -t:pack project.csproj` ) o eseguire l' `msbuild -t:pack` operazione in una cartella contenente un file di progetto.
