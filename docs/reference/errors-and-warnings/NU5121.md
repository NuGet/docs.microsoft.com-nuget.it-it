# <a name="nuget-warning-nu5121"></a>Avviso NuGet NU5121
<pre> 'content' assets will not be available when the package is installed after the migration.</pre>

### <a name="issue"></a>Problema

Il pacchetto contiene un file in una `Content` cartella. Questi asset non saranno disponibili se il pacchetto viene installato in un tipo di riferimento del pacchetto di progetto.


### <a name="solution"></a>Soluzione

Richiedere all'autore del pacchetto di correggere il pacchetto in modo che non contenga pi√π un `Content` cartella. Per altre informazioni, vedere la [documentazione di NuGet](../../consume-packages/migrate-packages-config-to-package-reference.md).
